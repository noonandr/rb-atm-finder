#Rebank ATM finder
*ATM finder exercise for Rebank. Lists the top 10 closest ATM locations based on the visitors geolocation.*

## Installation

cd into root folder:

```
$ cd rb-atm-finder
```

Install npm modules:


```
$ npm install
```

## Run Server

To run the server:

```
$ npm run serve
```
